# POCO F7 Ultra (miro) — Unified Support Hub

Welcome to the central support repository for the **POCO F7 Ultra (miro)**. This space is maintained by **getAOSP** and serves as the official hub for tracking bugs, collecting logs, and hosting community discussions for our custom ROM projects.

Currently, this hub officially supports the **v1.0 Initial Unofficial Releases** of:
* 💿 **LineageOS** (Android 16 Base)
* 💿 **Lunaris AOSP** (Android 16 QPR2)

## 🚨 Essential Rules Before Posting

Before opening an issue or asking for help, make sure you meet these requirements. Threads that ignore these rules will be **closed immediately**.

1. **Clean Flash Only (v1.0):** You **must** perform a full Factory Reset (`Format Data`) in recovery before reporting bugs on our v1.0 releases. Bugs occurring on "dirty-flashed" systems will not be triaged.
2. **Provide Logs:** For system crashes or bootloops, attach a log file captured via **MatLOG** or standard ADB logcat. A text description like *"Bluetooth doesn't work"* without a log is impossible to debug.
3. **Be Respectful:** This is a community-driven project built on open-source contributions.
